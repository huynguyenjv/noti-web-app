Gửi qua ingest API POST /api/v1/notification. Chọn channels: PUSH (fan-out mọi máy), INAPP (vào hộp thư 🔔), EMAIL/SMS/ZNS (skeleton). Đặt type = PAYMENT_FAILED để kích OUTAPP nudge — bật Realtime (góc trên) trước để nhận qua SSE realtime; không bật thì nudge đi push.
